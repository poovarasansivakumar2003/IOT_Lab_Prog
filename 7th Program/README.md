## 🚀 Program 7

### 📋 Objective
Arduino program to drive a DC motor and a stepper motor. 

---

### 🔌 Pin Configuration (Example)

#### DC Motor (via L298N or L293D)

- IN1 → Pin 8
- IN2 → Pin 9
- EN1 (PWM) → Pin 10

#### Stepper Motor (via ULN2003)

- IN1 → Pin 4
- IN2 → Pin 5
- IN3 → Pin 6
- IN4 → Pin 7

Use external power (not USB) for motors to avoid brownouts.
